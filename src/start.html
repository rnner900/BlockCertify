What is BlockCertify?

BlockCertify is a project with the aim of implementing certificates based on the Ethereum Blockchain. Therefore we used solidity to develop smart contracts and Bootstrap and jQuery for the UI. BlockCertify provides functionality to create courses, add and remove participants to these course and to issue certificates. All certificates assigned to the linked address are displayed to the user. Certificates and courses of other users can be received with the provided search functionality.


Getting Started with BlockCertify?

To get started you could do the following steps:
1. Create a Course
2. Display created Courses and display their Details by clicking on it
3. Add participants to your Course and then Issue Certificates for participants
4. Use the search functionality by searching for a users public address
5. Switch to an address in Metamask that you created a course and certificate for (Importing an account)
