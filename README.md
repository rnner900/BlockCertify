## Requirements:
* NodeJS
* Ganache

## Installation:
Navigate to project in terminal. 

run `npm install`
run `npm install -g truffle`

## Run application:
Make sure ganache is running (on HTTP://127.0.0.1:7545)
run `truffle migrate --reset`
run `npm run dev`
