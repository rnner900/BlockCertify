## Requirements:
* NodeJS
* Ganache

## Installation:
Navigate to project in terminal. 

1. run `npm install`
2. run `npm install -g truffle`

## Run application:

1. Make sure ganache is running (on HTTP://127.0.0.1:7545)
2. run `truffle migrate --reset`
3. run `npm run dev`
